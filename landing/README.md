# Session Cove – Landing Site

This is the marketing / landing page for Session Cove.

Tech:
- Next.js (App Router)
- TypeScript
- Tailwind CSS

Sections to implement on `/`:

1. **Header**
   - Logo on the left (Session Cove icon + wordmark placeholder).
   - Social links (X, Instagram, TikTok) on the right.

2. **Hero**
   - Background: gradient / wave using Session Cove brand colors.
   - Centered glass card with:
     - Badge: "WhatsApp-first · Private beta coming soon"
     - H1: "Turn WhatsApp chats into clean, reliable sessions"
     - Subheadline: short description for solo trainers and tutors.
     - Waitlist form:
       - Email (required)
       - "What do you do?" dropdown (trainer, tutor, etc.)
       - WhatsApp number (optional)
       - Primary button: "Join the early access list"
     - Small text: "No spam. Early access invitations and product updates only."

3. **Who it’s for**
   - Section title: "Built for solo trainers and tutors who live in WhatsApp"
   - Three cards:
     - Personal trainers & coaches
     - Tutors & teachers
     - Specialist instructors

4. **How it works**
   - Title: "How Session Cove fits into a normal day"
   - Short intro line.
   - 3 steps:
     1. Set your availability
     2. Share your booking link
     3. Let WhatsApp handle the follow-ups

5. **FAQ**
   - 3 questions:
     - When will Session Cove be available?
     - Will Session Cove work in my country?
     - Is Session Cove another chat app?
   - Short, clear answers.

6. **Footer**
   - © Session Cove, 2025
   - Links: X, Instagram, TikTok, optional email.

Design notes:
- Mobile-first layout. On desktop, use max-width container with more spacing.
- Use a subtle glassmorphism effect for the hero card (background white/10, border, shadow, backdrop blur).
- Keep everything fast and accessible: good contrast, readable text.
